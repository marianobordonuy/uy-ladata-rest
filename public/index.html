📡 ¿Qué ofrece?

Features

  • 💵 Cotizaciones USD/UYU en tiempo real desde múltiples fuentes (Real-time FX rates)
  • ⛽ Precios oficiales de combustibles ANCAP (Official fuel prices)
  • 🔁 Conversor de divisas USD ⇄ UYU (Currency conversion)

📘 Endpoints principales

Main Endpoints

  • GET /api/currency/all – Lista de fuentes (All providers)
  • GET /api/currency/live – Cotizaciones actualizadas
  • GET /api/convert – Conversión de divisas
  • GET /api/fuel – Precios de combustibles

🔐 Uso con API Key

Usage with API Key

Incluí tu API key en cada request como parámetro:

GET /api/currency/live?apikey=your_key

Ejemplo con curl:

curl "https://yourapi.com/api/convert?amount=100&from=USD&to=UYU&source=itau&apikey=your_key"
