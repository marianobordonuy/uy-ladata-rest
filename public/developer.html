Quickstart

Hacé tu primer request en segundos.

curl -s \
  "https://api.uyu.exchange/api/currency/live?apikey=TU_API_KEY"
fetch("https://api.uyu.exchange/api/currency/live?apikey=TU_API_KEY")
  .then(r => r.json())
  .then(console.log)
  .catch(console.error)
import requests
r = requests.get("https://api.uyu.exchange/api/currency/live", params={"apikey": "TU_API_KEY"})
print(r.json())

Autenticación

Usá tu apikey como query param: ?apikey=…. También soportamos header X-API-Key.

GET /api/currency/live HTTP/1.1
Host: api.uyu.exchange
X-API-Key: TU_API_KEY

Endpoints

GET /api/currency/live

Cotizaciones vivas de bancos y casas de cambio.

Parámetro Tipo Descripción
apikeystringTu clave de API (obligatorio).
sourcesstring[]Filtrar por fuentes (opcional).
formatstring"json" (default) o "csv".
Ejemplo:
GET https://api.uyu.exchange/api/currency/live?apikey=TU_API_KEY

Respuesta (JSON)

{
  "data": [
    { "name": "Itaú", "buy": 51.20, "sell": 53.50 },
    { "name": "BROU", "buy": 50.90, "sell": 53.10 }
  ],
  "updated_at": "2025-09-03T12:34:56Z"
}
Campo Tipo Descripción
dataarrayListado de fuentes con sus tasas actuales.
data[].namestringNombre de banco o casa de cambio.
data[].buynumberPrecio de Compra (UYU por USD).
data[].sellnumberPrecio de Venta (UYU por USD).
updated_atISO8601Timestamp de última actualización del conjunto.

Códigos de estado

  • 200 OK — respuesta con data.
  • 401 Unauthorized — API key inválida o ausente.
  • 429 Too Many Requests — superaste el rate limit.
  • 5xx Error en servidor — reintentá con backoff y caché corto.

Límites y caché

  • Rate limit sugerido inicial: 60 req/min por clave (consultanos si precisás más).
  • Cache recomendado en cliente: 30–60s para /currency/live.
  • Formato de números: punto decimal y 2–4 decimales.
